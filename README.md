# gdt-api

<h2>How to run local</h2>

- Clone from this repository

  ``` $git clone https://github.com/fauzilax/gdt-api.git ```

- change directory 

  ``` cd gdt-api```
 
- Run The Program

  ``` $go run main.go ```
